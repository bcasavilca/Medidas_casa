# Medidas Casa

Projeto de referência de medidas para ambientes e móveis domésticos.

## Descrição

Este repositório contém informações sobre medidas padrão para:
- Cozinha
- Quarto
- Sala
- Banheiro
- Móveis diversos

## Estrutura

- `/assets` - Imagens e recursos visuais
- `/dados` - Arquivos de dados com medidas
