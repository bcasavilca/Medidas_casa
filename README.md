# Medidas Casa

Repositório de referência de medidas para ambientes e móveis domésticos.

## Descrição

Projeto para documentar e organizar medidas de ambientes residenciais, facilitando o planejamento de reformas, projetos de interiores e compras de materiais.

## Ambientes de Referência

| Ambiente   | Medida Padrão                    |
|------------|----------------------------------|
| Cozinha    | Bancada: 60 cm profundidade      |
| Quarto     | Cama casal: 1,90 x 1,58 m       |
| Sala       | TV 50": 1,11 x 0,64 m           |
| Banheiro   | Vaso sanitário: 40 x 70 cm      |

1. Consulte as fotos para visualizar os ambientes
2. Use as medidas como referência para projetos
